退出当前知识库对话模式，并丢弃本次 OpenWiki 对话上下文。

## 步骤

### 1. 提示保存
如果本次 OpenWiki 对话中有未保存的问答，先提醒用户：`如需保存，先输入 /save <名称>；否则继续 /bye 将丢弃当前对话上下文。`

不要强制保存，不要阻塞式询问；只做提醒。

### 2. 退出对话模式
结束当前 OpenWiki 知识库对话模式，回到正常 Claude Code 交互。

### 3. 清除对话上下文
从执行 `/bye` 后开始，不再沿用本次 OpenWiki 对话模式中的临时问答上下文、最近问题、最近回答或未保存草稿。

后续如果用户再次使用 `/chat`，应重新读取 `wiki/index.md` 和相关页面，开启新的知识库对话上下文。

注意：这不会清空 Claude Code 的全局会话历史；如果用户想清空整个 Claude Code 上下文，需要使用 Claude Code 原生 `/clear`。
